Fire Fighting Robot

The main aim of this project is to develop a fire fighting robot that would detect the fire nearby it and extinguish that fire. A fire fighter robot is one that has a small fire extinguisher added to it. By attaching a small fire extinguisher to the robot, the automation put out the fire by human controlling. This report covers the design and construction of a robot that is able to sense and extinguish fire. This robot implements the following concepts: environmental sensing, proportional motor control. This robot processes information from its various sensors and key hardware elements via Arduino UNO. It uses flame sensors to detect the fire accident.
